Le collectif Co-demos développe différents logiciels open source

Ces outils permettent d’extraire, de consolider et de visualiser les informations.

Nous investissons dans l’innovation numérique au service de tous. La suite logicielle que nous avons développée est open-source et conçue de manière générique afin de pouvoir s'adapter à vos besoins. Cette suite peut être adaptée à d’autres bases de données que celle(s) présente(s) sur ce site, mais de nombreux autres aspects sont configurables. Les logiciels peuvent fonctionner indépendamment ou conjointement.



Open Scraper

Pour collecter des données publiées sur des sites

Fonctionnalités

  • Récupérer des données publiées sur des sites internet
  • Homogénéiser des données disparates
  • Ouvrir les données en maîtrisant leurs degrés d'ouverture (des données qui restent privées à de l’open data)
Voir sur Github


Solidata

Pour consolider et ouvrir des données

Fonctionnalités

  • Agréger des données provenant de sources ou de fichiers différents
  • Enrichir les données grâce à des données externes : géolocalisation, données administratives (SIRET), ...
  • Homogénéiser les tags des données agrégées
  • Ouvrir les données en maîtrisant leurs degrés d’ouverture (des données qui restent privées à de l’open data)
Voir sur Github


Apiviz

Pour valoriser des données de manière interactive

Fonctionnalités

  • Chercher des données via un moteur de recherche
  • Visualiser les données agrégées sous différentes formes : cartographie, datavisualisation, vue détaillée des données (fiches projets) …
Voir sur Github


TokTok

Microservice d'authentification

Fonctionnalités

  • Authentification d'utilisateurs via Json Web Tokens
  • Encryption RSA (optionnelle) et/ou identification anonyme
